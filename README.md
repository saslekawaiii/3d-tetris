# Tetris 3D

3D-версияигры Тетрис, написанная на C++ с использованием OpenGL и GLUT.

## Особенности игры

- Полноценная 3D-реализация механик Тетриса
- Вращение фигур по трём осям (X, Y, Z)
- Два режима отображения: вид сверху и сбоку(клавиши 1 или 0)
- Изначально игра стоит на паузе, чтобы ее запусить необходимо нажать кнопку S

## Управление игрой
| Категория управления | Действие | Основная клавиша | Примечание |
| :--- | :--- | :--- | :--- |
| Общие и старт | Пауза/Старт игры | `S` | Включает и выключает паузу |
| Общие и старт | Выход из игры | `ESC` | Немедленно закрывает игру |
| Управление фигурой | Перемещение фигуры | Стрелки `←` `→` `↑` `↓` | Направление зависит от вида камеры (`0`/`1`) и угла поворота сцены (`J`/`L`) |
| Управление фигурой | Вращение вокруг оси X | `X` | Работает только если игра не на паузе |
| Управление фигурой | Вращение вокруг оси Y | `C` | Работает только если игра не на паузе |
| Управление фигурой | Вращение вокруг оси Z | `V` | Работает только если игра не на паузе |
| Управление камерой | Вид сбоку (режим 0) | `0` | Камера смотрит сбоку |
| Управление камерой | Вид сверху (режим 1) | `1` | Камера смотрит сверху |
| Управление камерой | Поворот сцены влево | `J` | Поворачивает всю сцену вокруг вертикальной оси |
| Управление камерой | Поворот сцены вправо | `L` | Поворачивает всю сцену вокруг вертикальной оси |
| Управление камерой | Наклон сцены вперёд/назад | `I` / `K` | Наклон зависит от текущего угла поворота сцены |
| Управление камерой | Приблизить (зум) | `F` | Уменьшает значение `zoom` |
| Управление камерой | Отдалить (зум) | `D` | Увеличивает значение `zoom` |
| Скорость игры | Увеличить скорость падения | `=` | Увеличивает `down_speed` (только не на паузе) |
| Скорость игры | Уменьшить скорость падения | `-` | Уменьшает `down_speed` (только не на паузе) |

## Быстрый старт

### Клонирование репозитория
```bash
git clone https://github.com/saslekawaiii/3d-tetris.git
cd 3d-tetris
 ```
### Сборка приложения MacOS
```bash
clang++ -std=c++17 -o Tetris3D mainer.cpp shape.cpp -I/opt/homebrew/include -L/opt/homebrew/lib -framework OpenGL -framework GLUT
```
